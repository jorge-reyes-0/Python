# 🐍 Python — WGU Coursework

Welcome to my Python coursework repository for **Western Governors University**! This repo contains code, notes, and exercises built while working through WGU's Python courses. From the basics of data types all the way to network automation with SSH — it's all here. Let's get into it! 💻

---

## 📋 Overview

This repo covers:

- 🧱 **Python fundamentals** — data types, control flow, functions, and string formatting
- 📦 **Data structures** — lists, dictionaries, tuples, and sets
- 📂 **File I/O** — reading, writing, and comparing files
- 🌐 **Network automation** — SSH automation with Paramiko and Netmiko
- 🧩 **Modules & packages** — working with pip, the standard library, and third-party libraries
- 🐛 **Debugging** — error types and how to squash them
